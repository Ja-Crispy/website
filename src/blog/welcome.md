so this exists now.

i wanted a place to write about the things i'm working on — ml research, weird optimization problems, the occasional existential crisis about why programs are harder to find than neural networks. the usual.

## what you'll find here

technical writeups, mostly. each post borrows the voice of a different piece of fiction because life's too short for dry academic prose. expect hitchhiker's guide references next to fitness landscape data, tolkien metaphors alongside damage curves. the data stays rigorous, the tone does not.

topics i'm thinking about:

- **optimization geometry** — why some spaces are searchable and others aren't
- **program synthesis** — evolving SUBLEQ programs and the mesa topology
- **agentic systems** — what i'm building at work, when i can talk about it
- **random experiments** — whatever catches my attention

## the site itself

vanilla html/css/js. no frameworks. python script converts markdown to html. four color palettes that rotate on reload. a command palette (ctrl+k) with some secrets hidden in it. a tiny creature at the bottom of the page that's learning to explore via Q-learning.

i built this because i wanted to, and because "just use a template" is boring.

## what's next

the first real post is about SUBLEQ and mesa topology — why finding programs is harder than training the interpreters that run them. it's hitchhiker's guide themed. after that, who knows.

if you're reading this, hi. poke around. try the command palette. click things.
